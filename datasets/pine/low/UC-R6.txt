Use Case Name	Delete message
Summary	The user selects a message and deletes the mail message
Actor	Pine user
Pre-condition	The user logs in to the pine system.
Use Case ID	UC.R.2
Description	1.	The system displays a listing of all the available mail messages.
The user navigates to a folder.
The user selects a message and deletes it.
The system deletes the message and displays a listing of all the remaining messages.
Alternatives	If the user does not select any message and tries deleting, then the system displays a message asking the user to select at least one message.
Post-condition	The selected message is deleted.
